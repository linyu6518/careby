# Careby 网站：旧版 → 新版 更新说明

本文档对比**根目录（旧版）**与 **`careby/` 目录（新版）**的差异，说明从旧到新做了哪些更新。

---

## 一、项目位置与运行方式

| 项目 | 旧版 | 新版 |
|------|------|------|
| 目录 | 仓库根目录 | `careby/` 子目录 |
| 本地启动 | 根目录下 `npm run dev`（默认 http://localhost:5173） | `cd careby && npm run dev`（若 5173 被占用则用 http://localhost:5174） |

---

## 二、HTML 与基础配置

### 1. `index.html`

| 项目 | 旧版 | 新版 |
|------|------|------|
| viewport | `maximum-scale=5.0, user-scalable=yes` | 同上，并增加 **`viewport-fit=cover`**（适配刘海屏/安全区） |
| 字体加载 | **Inter** + **Noto Sans SC**，通过 JS 在 `window.load` 后异步加载 | **Plus Jakarta Sans**，通过 `<link rel="preload">` 加载，并设置 `fonts-loaded` class |
| 字体注释 | "Text fonts - Load after page load..." | "Text fonts - Plus Jakarta Sans" |

### 2. 全局样式 `src/index.css`

| 项目 | 旧版 | 新版 |
|------|------|------|
| 默认字体 | 系统字体，`.fonts-loaded` 后为 **Inter, Noto Sans SC** | 直接使用 **Plus Jakarta Sans**，`.fonts-loaded` 后仍为 Plus Jakarta Sans |
| 移动端触控 | 所有 `button, a, input, select, textarea` 最小 44×44px | 仅对 **主要控件**（`button:not([class*="inline"]), nav a, .min-touch`）设 44×44；增加 **`.inline-touch`**、**`.pb-safe`**、**`.bottom-safe`** 等，避免误伤行内链接 |
| 安全区 | 无 | 增加 **`env(safe-area-inset-bottom)`** 相关样式（`.pb-safe`、`.bottom-safe`），适配带刘海/底栏设备 |
| 横向滚动 | 无 | `body { overflow-x: hidden }`，防止移动端横向溢出 |

### 3. Tailwind 配置 `tailwind.config.js`

| 项目 | 旧版 | 新版 |
|------|------|------|
| 字体 | 未在 theme 中扩展 | **fontFamily.sans** 增加 **Plus Jakarta Sans** 及系统字体回退 |

---

## 三、首页内容与结构（App.tsx）

新版 `careby/src/App.tsx` 约 **5735 行**，旧版 `src/App.tsx` 约 **4293 行**；新版在文案、区块和交互上都有较大调整。

### 1. Hero 首屏

| 项目 | 旧版 | 新版 |
|------|------|------|
| 主标题 | **"Care by People Who Truly Care"**（headlinePrefix + headlineSuffix） | **"Care by Trust, Experts, Love."**（headlineSuffix 为空，仅 headlinePrefix + 新文案） |
| 副标题 | 仅一条 subheadline | 增加 **subheadlineStrong**（强调 AI 诊断、当日医生、上门护理等） |
| 眉标/说明 | 无 | **eyebrow**：`"Toronto · Mandarin & Cantonese · Doctor-Reviewed · PHIPA Compliant"`（中英粤、医生审核、PHIPA 合规） |
| 信任标签 | **trustBadges**（6 条长文案） | 保留 trustBadges，并增加 **trustChips**（如 "No referral needed"、"Same-day virtual doctor" 等短标签） |
| CTA | 主按钮 + "Call Us" | 增加 **secondaryConsult**（"Not sure? Talk to us first"）、**plansLink**（"See all plans ↓"） |

### 2. “谁需要我们”/痛点区块（Pain Points）

| 项目 | 旧版 | 新版 |
|------|------|------|
| 区块标题 | "Who We Serve" / "We Understand Your Challenges" | **"Who Careby is built for"** / **"Three gaps the system left open."** + **"We close all of them."**（titleEm） |
| 内容结构 | **audience1 / audience2 / audience3** 三张卡片（企业家、居家老人、健康关注者），每张带图片、标题、副标题、问题列表 | 改为 **problems** 数组：3 个“系统缺口”故事（situation / fear / answer / answerEm），更偏叙事与解决方案，无旧版三受众卡片 |
| 区块逻辑 | 按受众分类展示痛点 | 按“系统缺口 + 我们如何弥补”叙事，并衔接到下方 **We close all of them** 步骤区 |

### 3. 新增区块（仅新版）

以下区块在旧版中不存在，为新版独有：

- **We close all of them（WeCloseAllSection）**  
  - 眉标："Simple by design"  
  - 标题："We close all of them."  
  - 四步流程：Choose your plan → Meet your coordinator → We match & brief your team → You stay informed  

- **What Careby means（CarebyMeaningSection）**  
  - 三个关键词：**Care by Experts / Love / Trust**  
  - 各自配图标（听诊器、心、盾牌）和简短定义，强调“专家、关爱、信任”  

- **Why Careby exists（WhyCarebyExistsSection）**  
  - 眉标："Why Careby exists"  
  - 标题："The gaps are real." / "So is the fix."  
  - 数据卡片：如 650 万加拿大人无家庭医生、候诊时间可减少最多 90%、加拿大在 11 国中候诊最长、仅 43% 能当日看诊等，并带来源说明（Statistics Canada、GoToDoctor、Commonwealth Fund 等）  
  - 视觉：深色渐变背景 + 简单动效（线条、光点等）  

### 4. 其他内容与结构差异

- 新版中部分区块被**注释掉**（未渲染）：如 `NewSection`、`PlatformSection`、`ServiceSection`、`TechnologySection`、`FinalCTASection`。  
- 实际渲染顺序大致为：Hero → WhoWeServe → PainPoints → WeCloseAll → CarebyMeaning → Partners → Impact → MembershipTiers → FAQ → WhyCarebyExists。  
- 中文（含繁体）文案与结构随英文一起改版，例如 **"康伴 为何存在"**、**"我们全部解决"**、**"简单易行"** 等，与上述新区块一一对应。

## 四、部署与工程（仅新版具备）

| 项目 | 旧版 | 新版 |
|------|------|------|
| Vercel | 根目录无 `vercel.json` | **careby/vercel.json**：配置 build、rewrites、安全与缓存 headers（含 gzip/br、静态资源长期缓存等） |
| 部署脚本 | 无 | **careby/deploy.sh**、**careby/package-for-client.sh** 等 |
| 文档 | 根目录有部分 MD | careby 内另有 **DEPLOYMENT_GUIDE.md**、**PRE_DEPLOYMENT_CHECKLIST.md**、**README-FOR-CLIENT.md** 等 |

## 五、总结对照表

| 类别 | 主要更新点 |
|------|------------|
| **字体与排版** | Inter + Noto Sans SC → **Plus Jakarta Sans**；Tailwind 与 CSS 同步更新 |
| **移动端与安全区** | viewport-fit=cover；safe-area 样式；更精细的触控目标与横向滚动控制 |
| **Hero** | 新主标题 "Care by Trust, Experts, Love."；eyebrow、trustChips、secondaryConsult、plansLink |
| **痛点/受众** | 从“三类受众卡片”改为“三个系统缺口 + 我们如何弥补”的叙事结构 |
| **新首页区块** | We close all of them（四步流程）、What Careby means（Experts/Love/Trust）、Why Careby exists（数据与来源） |
| **部署与工程** | 新版在 careby 目录提供 Vercel 配置与部署/打包脚本 |

如需对某一块做更细的文案或代码级对比，可以指定文件或区块名称继续整理。
